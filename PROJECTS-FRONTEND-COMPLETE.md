### 🔗 Backend API (готов):

#### Проекты:
- `GET /api/projects` - Список с фильтрацией ✅
- `GET /api/projects/:id` - Детали + статистика ✅
- `POST /api/projects` - Создание ✅
- `PUT /api/projects/:id` - Обновление ✅
- `DELETE /api/projects/:id` - Удаление ✅

#### Квартиры:
- `GET /api/apartments` - Список с фильтрацией ✅
- `GET /api/apartments/:id` - Детали ✅
- `POST /api/apartments` - Создание ✅
- `POST /api/apartments/bulk` - Массовое создание ✅
- `PUT /api/apartments/:id` - Обновление ✅
- `DELETE /api/apartments/:id` - Удаление ✅
