### Backend API ✅

**Endpoints:**
- `GET /api/bookings` - Список бронирований с фильтрацией
- `GET /api/bookings/:id` - Детали бронирования
- `POST /api/bookings` - Создание бронирования
- `PUT /api/bookings/:id` - Обновление статуса (подтвердить/отменить)
- `DELETE /api/bookings/:id` - Удаление бронирования

**Ключевые функции:**
- ✅ Автоматическое изменение статуса квартиры (AVAILABLE → RESERVED)
- ✅ Валидация доступности квартиры
- ✅ Проверка что клиент принадлежит брокеру
- ✅ Автоматическое истечение брони по таймеру
- ✅ Транзакционная безопасность (бронь + квартира в одной транзакции)
- ✅ Проверка что нет активной брони на квартиру
- ✅ RBAC - брокеры видят только свои брони

**Статусы бронирования:**
- `PENDING` - Ожидание подтверждения
- `CONFIRMED` - Подтверждена
- `CANCELLED` - Отменена
- `EXPIRED` - Истекла (автоматически)

## 📊 Workflow бронирования:

### 1. Брокер открывает шахматку квартир
- Видит доступные квартиры (зеленые)
- Кликает на квартиру
- Нажимает "Забронировать"

### 2. Форма бронирования
- Выбирает клиента из своего списка
- Устанавливает срок брони
- Добавляет заметки
- Нажимает "Забронировать"

### 3. Система автоматически:
- Создает бронирование
- Меняет статус квартиры на RESERVED
- Квартира становится желтой в шахматке
- Устанавливает таймер истечения

### 4. Брокер в списке броней:
- Видит все свои бронирования
- Таймер показывает оставшееся время
- Может подтвердить → квартира остается забронированной
- Может отменить → квартира возвращается в AVAILABLE

### 5. Автоматическое истечение:
- Если срок истек → статус EXPIRED
- Квартира автоматически освобождается
- Снова доступна для бронирования

## 🔗 Связи в системе:

### Бронирование связывает:
1. **Клиент** → кто бронирует
2. **Квартира** → что бронируется
3. **Брокер** → кто создал
4. **Проект** → где находится (через квартиру)

### Автоматические обновления:
- Бронь создана → квартира RESERVED
- Бронь подтверждена → квартира RESERVED
- Бронь отменена → квартира AVAILABLE
- Бронь истекла → квартира AVAILABLE
- Бронь удалена → квартира AVAILABLE
