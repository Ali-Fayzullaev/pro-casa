## 🎨 TOAST УВЕДОМЛЕНИЯ:

### Типы уведомлений:

#### Успех (default):
```typescript
toast({
  title: "✅ Успех!",
  description: "Операция выполнена успешно",
})
```

#### Ошибка (destructive):
```typescript
toast({
  title: "❌ Ошибка",
  description: "Что-то пошло не так",
  variant: "destructive",
})
```

#### С продолжительностью:
```typescript
toast({
  title: "🎉 Сделка оформлена!",
  description: "Квартира успешно продана",
  duration: 5000, // 5 секунд
})
```
